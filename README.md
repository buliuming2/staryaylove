# 豆芽人联盟：星芽之约

《豆芽人联盟：星芽之约》是一款使用 Python 打造的文本型 Galgame。

在这里，你将扮演一位误入“豆芽人联盟”的普通少年，与五位性格迥异的少女相遇——

- 🎨 千日坂芷芽：安静温柔的绘画少女，用画笔记录每一寸即将消逝的时光。
- 🍰 渚浔笑辞：元气满满的料理少女，笑容如蛋挞般甜蜜，却也有独自哭泣的夜晚。
- 💻 小鸟游刘网：沉默的天才程序员，被全班排挤孤立，代码是她与世界唯一的对话方式。
- ⚓ 加比镜岚：向往大海的航海少女，眼里藏着地平线，梦想是驾驶帆船去看看世界的尽头。
- 🍦 棱斯新耀羽：耀眼如太阳的冰棍少女，笑容比草莓味还甜，为了妈妈的医药费努力发光。

每一次对话，每一个选择，都会影响她们的命运与结局。  
是成为守护她们的伙伴，还是无意中将她们推入黑暗？  
教室里、天台边、厨房中……那些平凡日常里，藏着最深的羁绊。

支持 `Ctrl+T` 加速阅读，粉色文字渲染温柔氛围，  
多结局分支设计，带你体验欢笑与泪水交织的青春物语。

**豆芽人联盟，星芽之约——你，愿意为谁停留？**
